Individual Project

Title:World Happiness Report

Data Source:Kaggle.com

Link to data: https://www.kaggle.com/unsdsn/world-happiness

Description In this project I wanted to calculate the happiness score for each country and visualize them using folium/Tableau based on the ranking. The Happiness Score is calculated by the following factors:

  1. GDP per capita
  2. Healthy Life Expectancy
  3. Social support
  4. Freedom to make life choices
  5. Generosity
  6. Corruption Perception

Visualization Idea:

  • Geographical map with different countries using Folium

  • Ranking bar with different color

  • Horizontal bar with years on bottom

    • Output: Through this analysis, I want to show the ranking of different countries based on different factors.
